MERSD Teacher Tools

A collection of simple tools and utilities for teachers and school staff. Select a tool below to get started. If you have an idea for a tool, please reach out to Chris Dwyer.

QR Code Generator
Generate MERSD-branded QR codes for easily sharing links and resources.
OCR Visual Read Aloud
Scan images containing text and have it read aloud. Works best with clear, high-contrast images containing only text.
Google Drive Voice Recorder
Simple audio recording tool that saves audio files directly to Google Drive.
Lunch Choice Board
Generate a custom lunch choice board with your class roster. Get a reusable link that you can bookmark in your browser.
